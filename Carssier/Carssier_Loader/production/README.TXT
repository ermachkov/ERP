Скрипт client.sh предназначен для запуска сервера* Carssier и автоматического запуска графической оболочки**
Скрипт server.sh предназначен для запуска сервера* Carssier

-----------------------------------------------------------------
* - сервер будет запущен на порту 1572, для служебных целей также будут заняты 2 порта 1573 и 1574

** - за запуск графической оболочки отвечает скрипт ~/.saas/app/bin/startui.sh. 
Отредактируйте при необходимости путь к браузеру Chrome. 
Внимание!!! 
При закрытии графической оболочки, также будет остановлен сервер Carssier.
Для корректной работы программы необходим браузер Chrome версии 20 и выше
